System Overview

The Simplex Method Calculator is an educational web application that walks users through linear programming problems using the Simplex algorithm. It visualizes tableau transformations, pivot choices, and produces the final optimal solution for maximization problems.

This system is intended for students, instructors, and researchers who want a clear, interactive way to study and demonstrate linear programming.

Objectives

  • Provide interactive LP solving
  • Show step-by-step tableau visuals
  • Help students with downloadable notes
  • Work offline and maintain a beautiful UI

System Features

  • ✔ Step-by-step tableau
  • ✔ Auto pivot detection
  • ✔ Dark/light mode
  • ✔ Offline capable
  • ✔ Modules & notes
  • ✔ Student team
  • ✔ Responsive design

Technologies Used

HTML5 • CSS3 (Glassmorphism + Animations) • JavaScript • solver.js • LocalStorage

System Flow

[User Input] → [solver.js] → [Tableaus] → [Final Solution]

How to Use

  1. Enter the objective function (e.g., 1.5x + 1.2y).
  2. Add constraints and set RHS values.
  3. Click Solve to see step-by-step tableaus.
  4. Download module notes for study.
  5. Switch themes with the toggle in the header.
